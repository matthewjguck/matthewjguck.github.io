What is an Inkling?

Inklings are a tangible reminder that someone is thinking about you.

They turn kind, passing thoughts into meaningful connection between friends.

User Generated Inklings

How does Inklings work?

Inklings consists of a network of physical receipt printers and a web app.

Sending

Senders create their message on the Inklings web app, and send a notification to their friend's Stanford University email.

The message editor allows users to write with unique fonts, dither images for B&W printing, and add fun stickers via GIPHY's sticker API.

Receiving

Recipients of an Inkling receive an email notification prompting them to go to an Inklings printer and print their message.

Once they sign in and click print, Inklings will ensure that device's location is within 10m of a printer. If it is, the message will print within 3 seconds.

Technical

Each printer is connected to a Raspberry Pi running a custom print server, which continuously polls our Supabase backend for new print jobs and processes them automatically.

The Problem

Our team originally set out to address the problem of connecting with loved ones at a distance.

Every existing way of reaching someone forced a tradeoff. Mailed letters and handwritten cards feel intentional, but they're slow and take real commitment. Texts, DMs, and social feeds are fast, but they're often passive — easy to send without much thought, and just as easy to forget. Nothing let you send something quick that still felt like it meant something.

The Solution

Inklings is a new form of communication that is more intentional than a text, but less commitment than a posted letter.

Inkling

[ink-ling] · noun

1. a small, tangible reminder that someone is thinking about you

Users compose an "Inkling" on the web app and send it to a friend's Stanford SUNet ID. It shows up as a surprise — printed out on one of a network of receipt printers placed around campus, ready for the recipient to find, rip off, and keep.

Inklings Web App

1. Start an Inkling

Open the web app and choose to send or print a new Inkling.

2. Pick a Friend

Address it to their Stanford SUNet ID — no accounts or contact syncing needed.

3. Add a Note

Write a message, drop in a photo or sticker, and send it off.
